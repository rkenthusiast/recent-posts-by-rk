== Description ==

Recent Posts by RK is a WordPress plugin that allows you to display recent posts in a customizable widget. You can select categories, tags, and the number of posts to display.

== Installation ==

1. **Install Dependencies**: Before uploading the plugin, make sure to install any dependencies required by running the following command in the plugin directory:
   ```bash
   npm install
2. Upload the `recent-posts-by-rk` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Go to 'Appearance' > 'Widgets' to add the Recent Posts by RK widget to your sidebar.

== Frequently Asked Questions ==

= How do I customize the recent posts display? =

You can customize the display through the widget settings. Go to 'Appearance' > 'Widgets', and click on the Recent Posts by RK widget to see the available options.

= Can I display posts from specific categories or tags? =

Yes, you can select specific categories or tags in the widget settings.
